💡 Flashlight Pro — фонарик с режимом SOS
Многофункциональный фонарик с поддержкой режимов: постоянный свет, стробоскоп, плавное затухание, сигнал SOS (азбука Морзе), и настройка яркости.
Реализован на 7 языках программирования для демонстрации подходов к управлению светом и звуком.

🧠 Концепция
Flashlight Pro — это не просто фонарик. Это инструмент для экстренных ситуаций и повседневного использования:

✅ Режим «Постоянный свет» — стандартное освещение с регулируемой яркостью.

✅ Режим «SOS» — мигание сигналом бедствия (три коротких, три длинных, три коротких) с паузой.

✅ Режим «Стробоскоп» — быстрое мигание с настраиваемой частотой.

✅ Режим «Плавное затухание/нарастание» — эффект свечи или «ночник».

✅ Настройка яркости — от 0 до 100% (в GUI-версиях слайдер, в консоли — команды).

✅ Звуковое сопровождение (в некоторых версиях) — пищалка при включении/выключении.

✅ Сохранение настроек — запоминание последнего режима и яркости.

✅ Визуальный индикатор — отображение текущего режима и параметров (для GUI).

🚀 Как запустить
Для GUI-версий требуется установка соответствующих библиотек. Консольные версии работают в терминале с использованием ASCII-анимации или системных вызовов (для мигания экрана или звука). В реальном проекте управление аппаратным фонариком (например, через GPIO) может быть добавлено.

bash
# Python (Tkinter)
pip install tk  # обычно предустановлена
python flashlight_python.py

# C++ (Qt)
qmake && make
./flashlight_cpp

# Java (Swing)
javac flashlight_java.java && java flashlight_java

# C# (WPF)
dotnet run

# Go (консоль)
go run flashlight_go.go

# Rust (консоль)
cargo build --release && ./target/release/flashlight_rs

# JavaScript (Node.js)
node flashlight_js.js

🔮 Расширенные функции
Режим «Световой сигнал» — пользовательский шаблон мигания (например, азбука Морзе).

Автоматическое отключение по таймеру (экономия батареи).

Поддержка RGB (для светодиодных лент) — в планах.
